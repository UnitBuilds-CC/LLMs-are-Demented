👾 Welcome to The Inference Arena!

Welcome, Engineer. Today, you will solve a standard crossword puzzle. The catch? You must work within the exact architectural constraints of a running Large Language Model (LLM).

💡 TUNING ADVICE: Playing under constraints is highly volatile. You are strongly encouraged to tune your parameters (manually adjusting sliders or selecting hardware presets) to find a configuration that makes the game winnable for you!

💥 THE MECHANICS OF FRUSTRATION:

  • Context Window (Ctokens): Your active memory queue. The system only remembers your last N cell edits. If you fill in more than N letters, the oldest letters you typed will immediately hallucinate/mutate!
  • KV-Cache Expiry (τ): The board is divided into 4 quadrants. Each quadrant's cache will decay over time. If a quadrant is left untouched for more than τ seconds, the cache expires and wipes that entire quadrant blank! Edit a cell in a quadrant to refresh its cache.
  • Temperature (T): Controls the nature of the mutations. Lower temperatures cause letters to drift into visually similar characters (e.g. A ➔ 4, E ➔ 3). Higher temperatures produce complete symbolic and emoji chaos (e.g. %, 🤖).

Can you type fast enough, manage your sliding context window, maintain active caches in all quadrants, and trigger a 100% accurate ⚡ RUN INFERENCE evaluation?

1. WEIGHTS are the learned numerical parameters inside neural network layers.

2. TOKEN is the fundamental unit of text processed by a transformer.

3. MODEL is a trained system of weights and biases representing patterns in data.

4. EPOCH is one complete pass through the entire training dataset.

5. BIAS is the additive parameter offset applied in dense layers alongside weights.

6. ATTENTION is the core transformer mechanism that dynamically weights the relationships between input tokens.

7. LOSS is the error metric minimized during training.

7. LAYER is a repeating block of neural network computations (like attention or MLP).

8. PROMPT is the input instructions or query fed into an LLM to steer its generation.
